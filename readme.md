# TTS DISCORD BOT
A TTS bot to read discord messages

### REQUIREMENTS
- Node.js
- FFMPEG

### HOW TO INSTALL
1. Clone the project
2. Run `npm install`
3. Create a file called `settings.js` with this code:
```javascript
module.exports = {
    token: 'bot_token'
}
```
4. Run `node index.js` and enjoy! =D


### HOW TO USE
